Carbon Footprint Calculator 🌍

A web-based carbon footprint calculator that estimates annual CO₂ emissions based on:

Electricity usage (kWh/month)
Transport type
Flights per year

The calculator provides a detailed emission breakdown and categorizes the total footprint as Low, Moderate, or High.
